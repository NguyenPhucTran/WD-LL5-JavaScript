Step-by-Step Setup

Wiring JavaScript
Into Your Page

Before you write a single line of logic, you need JavaScript connected to your HTML. Follow these steps every time you start a new project.

  1. 01
    Create the file

    Make a new file named script.js in your project folder.

  2. 02
    Link it in HTML

    Add the script tag just before the closing </body> tag.

  3. 03
    Verify the connection

    Open DevTools → Console and confirm your message appears.

index.html
<body>
  <!-- your HTML here -->
 
  <script src="script.js"></script>
</body>
script.js
console.log("Welcome to CodeFest!");
⚠️
Common Gotchas
  • Script tag must go before </body>
  • Filename must match exactly — case-sensitive
  • Always save before refreshing
  • Refresh the browser to see changes

30-Minute Sprint

Build the Event Welcome Center

You're helping organize a large university technology conference. Attendees are arriving and you need a system that stores information and displays messages. Work through each challenge below.

01

Event Information

Create variables for your event and display each value with console.log().

let eventName = "CodeFest";
let speakerName = "Dr. Kim";
let roomNumber = 204;
let attendeeName = "Jordan";
 
console.log(eventName);
02

Personalized Greetings

Combine variables and strings to create custom welcome messages.

console.log(
  "Welcome " + attendeeName
  + " to " + eventName + "!"
);
 
console.log(
  "Your session is in Room "
  + roomNumber + "."
);
03

Build Functions

Wrap your messages inside functions and call them. Create at least two.

function welcomeGuest() {
  console.log("Welcome to " + eventName);
}
 
function displaySessionInfo() {
  console.log("Room: " + roomNumber);
}
 
welcomeGuest();
displaySessionInfo();
04

Alert Messages

Use alert() to send messages directly to the user. Notice how it differs from console.log().

// User sees this — not just the console
alert("Welcome to CodeFest!");
 
alert(
  "Session starts in Room " + roomNumber
);
💡 Add one alert per team member and watch the browser behavior.
05

Attendee Counter

Track how many attendees have checked in using a counter variable. Use ++ to increase the count.

💬 Discussion: Why might event organizers need attendance counts? Think about fire codes, catering, and planning. 
let attendeeCount = 0;
 
console.log("Count: " + attendeeCount);
 
attendeeCount++;
console.log("Count: " + attendeeCount);
 
attendeeCount++;
console.log("Count: " + attendeeCount);
